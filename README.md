# Endfield Damage Calculator

アークナイツ：エンドフィールドのダメージ計算機を作るためのプロジェクトです。

## 目的

オペレーターや敵のステータスをもとに、ダメージを計算できるようにすることを目的としています。

## 方針

最初から完璧な計算機を作るのではなく、まずは最小限の機能を実装します。  
その後、オペレーター、敵、レベル、スキル倍率などを少しずつ追加できる形にします。

## 最初に実装する範囲

- オペレーター情報の一部
- 敵情報
- レベル90固定のステータス
- 基本的なダメージ計算

## データ管理

データ管理には SQLite を使用します。  
データベースファイルは `.db` として管理します。

## 今後追加したい機能

- オペレーターレベルの変更
- スキル倍率の追加
- 敵の防御・耐性の追加
- 計算式の種類の追加
- UI の改善
